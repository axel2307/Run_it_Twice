# Poker Variance Simulator

Interactive Streamlit app to visualize variance in poker all-ins and the effect of running boards multiple times.

The simulator compares:

- Run it once
- Run it multiple times (2x, 3x, 4x)

and shows how variance changes while **EV remains the same**.

## Features

- Simulate thousands of poker all-ins
- Adjustable equity
- Adjustable pot size
- Adjustable rake
- Run it once vs multiple runs
- Distribution of results
- Downside risk curves
- Results expressed in buy-ins
